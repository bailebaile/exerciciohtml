Linguagens Server Side vs Client Side

As linguagens de programação web se dividem em dois tipos:

Server Side

Executadas no servidor antes de enviar conteúdo ao navegador. Processam dados, interagem com bancos de dados e geram conteúdo dinâmico.

Exemplos: PHP, Python, Ruby, Java, C#

Client Side

Executadas no navegador do usuário. Manipulam a interface, respondem a eventos e fazem atualizações dinâmicas na página.

Exemplos: JavaScript, HTML, CSS

Ambos os tipos são importantes no desenvolvimento web moderno, trabalhando juntos para criar sites interativos.
